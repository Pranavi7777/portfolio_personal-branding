
# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio showcasing my **skills, projects, certifications, achievements, and resume** as an aspiring **Data Scientist & AI/ML Enthusiast**.

🔗 **Live Demo:** https://portfolio-personal-branding.onrender.com/

## ✨ Features
- Responsive Design
- About Me
- Skills
- Projects
- Certifications
- Resume
- Contact Form

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- Python
- Flask
- Git & GitHub
- Render

## 🚀 Run Locally
```bash
git clone https://github.com/Pranavi7777/Portfolio.git
cd Portfolio
pip install -r requirements.txt
python app.py
```
